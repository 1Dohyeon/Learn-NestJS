# Study-NestJS

### clone

아래 과정을 통해서 프론트 파일을 불러온다.

```
git clone -b frontend --single-branch https://github.com/amamov/teaching-nestjs-a-to-z.git
```

```
cd teaching-nestjs-a-to-z.git
```

```
npm i
```

```
npm run dev
```

백엔드 폴더로 들어가서 `npm i` 실행 후 .env파일을 만들면 된다.
